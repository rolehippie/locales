## Requirements

- Minimum Ansible version: `2.10`

## Default Variables

### locales_available

List of available locales

#### Default value

```YAML
locales_available:
  - en_US.UTF-8
```

### locales_extra

Optional dict of additional locales

#### Default value

```YAML
locales_extra: {}
```

### locales_lang

Default language to use on the system

#### Default value

```YAML
locales_lang: en_US.UTF-8
```

### locales_packs

List of language packs to install

#### Default value

```YAML
locales_packs:
  - language-pack-en
  - language-pack-en-base
```
